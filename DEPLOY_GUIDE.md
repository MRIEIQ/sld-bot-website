# كيف ترفع الموقع على GitHub Pages

## الخطوات (بدون Git - طريقة الموقع)

### 1. أنشئ Repository جديد
1. ادخل على https://github.com/new
2. اسم الـ repo: sld-bot-website
3. اختر Public
4. اضغط Create repository

### 2. ارفع الملفات
1. داخل الـ repo اضغط Add file > Upload files
2. اسحب كل ملفات مجلد website:
   - index.html
   - privacy.html
   - terms.html
   - style.css
   - script.js
   - logo.png
3. اكتب Commit message: Add website files
4. اضغط Commit changes

### 3. فعل GitHub Pages
1. اضغط Settings في الـ    
2. من القائمة الجانبية اضغط Pages
3. تحت Branch اختر: main ثم / (root)
4. اضغط Save
5. انتظر دقيقة وحدث الصفحة

### 4. رابط موقعك
https://USERNAME.github.io/sld-bot-website/

### 5. ربط Privacy Policy ببوتك
1. ادخل Discord Developer Portal: https://discord.com/developers/applications
2. اختر بوتك (ID: 1289557315741159544)
3. اذهب لـ General Information
4. Privacy Policy URL: https://USERNAME.github.io/sld-bot-website/privacy.html
5. Terms of Service URL: https://USERNAME.github.io/sld-bot-website/terms.html
6. اضغط Save Changes
