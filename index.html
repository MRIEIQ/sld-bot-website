About

What is SLD Bot?

SLD Bot is a feature-rich Discord music bot built for communities that demand the best. Whether you're looking for crystal-clear music playback, personal playlists, audio effects, or handy server utilities — SLD Bot has you covered, 24/7.

🎵

Premium Music

Stream from YouTube, Spotify, and SoundCloud with high-quality audio and audio effects like bass boost, reverb, and equalizer presets.

📂

Personal Playlists

Create, save, and load personal playlists. Access your saved songs in any server with a single slash command, any time.

🌐

Multilingual

Commands

All Slash Commands

Every command is available as a Discord slash command. Just type / in any channel.

/playPlay a song or URL from YouTube/Spotify
/skipSkip the current song
/stopStop playback and leave the voice channel
/nowplayingShow the currently playing track
/queueDisplay the song queue
/shuffleShuffle the queue randomly
/repeatToggle repeat for the current song
/seekJump to a specific timestamp in the song
/volumeSet the playback volume (1–100)
/searchSearch for songs and pick from results
/lyricsFetch lyrics for the current or named song
/voteskipStart a vote to skip the current song
/removeRemove a specific song from the queue
/moveMove a song to a different position in the queue
/audio_effectsApply bass boost, reverb, treble, or EQ preset
/save_queueSave the current queue for later
/load_queueLoad a previously saved queue
/createplaylistCreate a new personal playlist
/playlistsView all your saved playlists
/loadplaylistLoad and play a saved playlist
/deleteplaylistDelete one of your playlists
/pingCheck bot latency and system performance
/serverShow detailed server information
/userinfoShow information about a user
/avatarGet the display avatar of a user
/setnickChange a member's nickname (Admin)
/movemeMove yourself to another voice channel
/prefixChange the bot's command prefix (Admin)
/setupSet up bot language for this server (Admin)
/calcEvaluate a math expression
/rollRoll a dice (e.g. 2d6)
/emojisList all server emojis
/helpShow the full help menu with all categories
/inviteGet the bot's invite link to add to your server
/voteVote for SLD Bot on Top.gg (every 12 hours)
Server Rules

Community Rules

By joining our server or using our bot, you agree to follow these rules.

01

Be Respectful

Treat all members with respect. Harassment, hate speech, or discrimination of any kind will result in an immediate ban.

02

No Spam

Do not flood channels with repeated messages, excessive pings, or irrelevant content. Use the appropriate channels.

03

No NSFW Content

All content must be suitable for a general audience. NSFW material is strictly prohibited outside of designated channels.

04
